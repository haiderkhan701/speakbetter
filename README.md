# SpeakBetter

Browser-based English speaking practice with live WebRTC audio rooms, synchronized transcripts, grammar corrections, AI coaching, and downloadable reports.

## Deploy

[![Deploy to Render](https://render.com/images/deploy-to-render-button.svg)](https://render.com/deploy?repo=https://github.com/haiderkhan701/speakbetter)

Click **Deploy to Render**, authorize the repository, and create the Blueprint service. Render will provide the public HTTPS link used by SpeakBetter's Invite button.

## Run locally

```bash
npm start
```

Open `http://localhost:4173`.
